Trabalhando com Listas

Listas Ordenadas:

A tag OL + a tag LI servem para fazer uma lista em ordem, como esta:

  1. Acordar;
  2. Tomar banho;
  3. Café da manhã;
  4. Faculdade;
  5. Almoçar;
  6. Relaxar;
  7. Lanchar;
  8. Estudar;
  9. Jantar;
  10. Academia;
  11. Dormir.

Para personalizar, se coloca o parâmetro type no ol.

Listas Não-Ordenadas:

Para uma lista não-ordenada, utiliza-se a tag UL.

  • Pão
  • Leite
  • Ovos
  • Queijo
  • Alface

Lista de Definição:

Para a lista de definições usa-se a tag DL, utiliza-se a tag DT para o termo e a tag DD para a descrição.

HTML:
Linguagem de marcação para a criação do conteúdo de um site.
CSS:
Linguagem de marcação para a criação do design de um site.
JavaScript:
Linguagem de programação para a criação da interatividade de um site.
