Principais Formatações

NEGRITO:

Nesta frase temos um termo em negrito, usando a tag B (não semântica).
Já nesta aqui, o termo em negrito utilizou a tag STRONG(semântica).

ITÁLICO:

Nesta frase, temos um termo em itálico usando a tag I (não semântica).
Já nesta outra frase, usamos a tag EM (semântica).

TEXTO MARCADO:

Podemos criar um texto marcado usando a tag MARK. E para qualquer marcação ficar do mesmo estilo, configura na head.

TEXTO GRANDE E PEQUENO:

Estamos criando um texto grande com a tag BIG (forma) e um texto pequeno com a tag SMALL neste parágrafo.

TEXTO DELETADO:

Textos excluídos devem ser lidos, mas não considerados. Usa-se a tag DEL.

TEXTO INSERIDO:

Podemos usar este para indicar que foi colocado depois. Neste caso, a tag semântica é a INS.
Para mudar apenas a forma, utiliza-se a tag U.

TEXTO SOBRESCRITO:

É usada a tag SUP para escrever coisas como x20+3.

TEXTO SUBSCRITO:

Usando a tag SUB, você escreve coisas como H2O.
